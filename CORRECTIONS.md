# Corrections

This is the public, append-only log of every approved correction to
the CanadaScanada corpus. Each entry records what was wrong, what is
now right, and (where applicable) who reported it.

This file is append-only. Post-launch, errors cannot be silently
fixed; every change is documented here.

## How to submit a correction

Use the form at `canadascanada.ca/feedback`, select "Correction" as
the submission type, and explain what should be changed and why.
Include a primary source URL if possible.

If your correction is accepted, an entry like the ones below will
appear in this file.

## Entry format

```
## YYYY-MM-DD · EVT-### · brief description of change

A short, plain-language paragraph explaining what was wrong, what is
now right, and (if applicable) who reported it. Source:
[outlet, date](URL).
```

## Pre-launch corrections

This file is initialized empty at the moment of public launch. The
project's working iterations included several internal corrections
(documented in the project's private working repository); from
launch forward, every change is logged here.
